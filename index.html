About the Workshop

This is a three day intensive workshop to provide attendees hands on experience on doing 16s rRNA sequencing analysis using R. While majority of teaching will be in R, we will also discuss best practices, study design and teach basics of unix tools needed to start an analysis from raw fastq files .

A rough overview of the topics covered is as follows:

  • How to design such an experiments
  • Basic read quality checks
  • Primer removal
  • Walk through the DADA2 pipeline
  • Options for taxonomic assignment
  • Using Phyloseq package to do conduct downstream analysis
  • Plotting community compositions, Alpha and Beta diversity
  • If time permits, basics for differential abundance analysis

    • Instructors
